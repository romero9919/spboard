개요

핵심 기능
---
1. AWS RDS에 MySQL을 통해 DB 생성, members 테이블을 통한 회원가입 및 로그인
2. 회원 권한 컬림이 'Admin'일 경우 관리자 권한 부여, 페이지 내부 회원 목록 관리 기능
3. 게시판 CRUD(Create, Read, Update, Delete) 기능 구현
4. 사진 파일 업로드 및 표시 기능 구현
